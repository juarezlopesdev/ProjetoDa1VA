===========================================================
  json-processor: Ferramenta de Processamento JSON em Haskell
===========================================================


DESCRIÇÃO BÁSICA DO PROJETO
-----------------------------

"json-processor" é uma ferramenta de linha de comando desenvolvida em Haskell, focada na manipulação e validação de arquivos JSON. O projeto foi criado para oferecer uma solução rápida, segura e eficiente para duas tarefas comuns no processamento de dados:

1.  Filtragem de Dados: Permite filtrar um array de objetos JSON com base em critérios de igualdade (ex: encontrar todos os objetos onde o campo "status" seja "ativo").

2.  Validação de Esquema: Permite validar se um arquivo de dados JSON está em conformidade com um esquema customizado, verificando a presença de campos obrigatórios e seus respectivos tipos de dados.

O objetivo da ferramenta é simplificar fluxos de trabalho que envolvem JSON, utilizando os pontos fortes da linguagem Haskell, como segurança de tipos e pureza funcional, para garantir operações confiáveis e previsíveis.
